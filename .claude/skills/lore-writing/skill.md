# Lore Writing Skill

All campaign prose - backstories, NPC bios, location descriptions, item lore, session recaps - should follow this style guide. Read the relevant reference before writing or rewriting anything.

## References

- **Style Guide** - `references/style-guide.md` - concrete DOs and DON'Ts distilled from the best fantasy lore writing
- **Sample Passages** - `references/sample-passages.md` - annotated examples from Dark Souls, Elden Ring, Rothfuss, Planescape, Curse of Strahd, and Mistborn

## When to Use

- Writing or rewriting character backstories
- Writing NPC descriptions or bios
- Writing location or item lore
- Writing session recaps or highlight text
- Any narrative prose for the campaign

## Key Principles (quick ref)

1. Write from the **aftermath**, not the event
2. Use the **gap between *was* and *is*** as your primary tool
3. Anchor emotion in **one strangely specific physical detail**
4. **Never name the emotion** - show the texture of it
5. **Vary sentence length** - short for weight, long for layered texture
6. Let the **voice match the source** - a journal sounds like a journal, not a narrator
7. **Leave the mystery open** - give consequence without cause
